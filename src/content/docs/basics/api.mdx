The View Transition API is a rather new API. In thi section you can find&hellip;

* &hellip;some [basic examples](/basics/examples/) of same-document and cross document view transitions.
* &hellip;information about the [structure of the pseudo-elements](/basics/pseudos/) of the API.
* &hellip;and about the [mechanisms of the default animations](/basics/default-animations/).
* &hellip;some background on [how to style view transitions](/basics/styling/).
* &hellip;details on how to use the [JavaScript API](/basics/javascript/) for same-document view transitions.
* &hellip;all there is to know about the [stacking behavior](/basics/hide-and-seek/) of view transitions.

While I add more pages, I'm also happy that I can offer some links to previous work on the `astro-vtbot` package[^1].

* An [Overview](https://events-3bg.pages.dev/jotter/api/) and a [Details](https://events-3bg.pages.dev/jotter/api/details/) article on the API
* An in depth [same-document example](https://events-3bg.pages.dev/jotter/api/example/)

[^1]: https://www.npmjs.com/package/astro-vtbot
